## asgn2.html
Main webpage that contains the WebGL canvas, sliders, buttons, FPS display, and loads all project scripts.

## asgn2.js
Main JavaScript file that controls WebGL setup, rendering, animation, matrix transformations, user interaction, and pig model logic.

## Cube.js
Defines the Cube class used to render cube-based body parts such as the body, head, arms, legs, eyes, and mouth.

## Cylinder.js
Defines the Cylinder class used to render the pig’s round snout as a non-cube primitive.

## Triangle.js
Contains the helper function used to draw 3D triangles, which is the base for rendering cubes and cylinders.
