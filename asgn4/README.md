## asgn4.html
Main webpage containing the WebGL canvas, buttons, sliders, and script imports.

## asgn4.js
Main JavaScript file that sets up WebGL, connects shader variables, handles UI controls, updates light/camera variables, and renders the scene.

## Cube.js
Defines the cube object, including vertex positions, UVs, normals, buffers, model matrix, normal matrix, and render function.

## Sphere.js
Defines the sphere object. Generates sphere vertices and normals using loops.

## Camera.js
Controls camera position, view matrix, and movement.

## Model.js
Loads and renders an OBJ model with lighting.

## models/
Contains OBJ model files used in the scene.

## textures/
Contains texture images used for the world.
